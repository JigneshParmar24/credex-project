# FAQ

## 1. How does BurnRate calculate savings?

BurnRate uses deterministic pricing rules and vendor pricing data to evaluate your current plans against your team's stated workflow and capability requirements. AI is only used for classification and summary generation — not for the savings calculations themselves.

---

## 2. Does BurnRate connect to my accounts?

No. The current version is fully self-reported. Users manually enter their tools, seats, and monthly spend.

Future versions may support direct integrations.

---

## 3. Which tools are supported?

Current support includes:
- ChatGPT
- Claude
- Gemini
- Cursor
- GitHub Copilot
- Windsurf
- v0
- Lovable
- OpenAI API
- Anthropic API

---

## 4. Is my audit data stored?

Not currently. The MVP version intentionally avoids authentication and persistent storage to keep the flow lightweight and frictionless.

---

## 5. What happens after the audit?

Users can optionally request a consultation with Credex to further optimize AI infrastructure spend, negotiate credits, or reduce API costs at scale.
